# Zorka

A small blog hosted on GitHub Pages.

**Live site:** https://noobistas.github.io/zorka/

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

## Add a post

1. Create `posts/your-slug.html` (copy `posts/welcome.html` as a template).
2. Add a link in the Writing list on `index.html`.
3. Commit and push to `main`.
